# Irrigation water voucher system
This is a full stack applicaiton deployed on cloudflare for purchasing vouchers for watering hours and redeeming them (getting the water supply to the respective farm that has purchased a voucher)

## Under construction
this application is still under construction and pre MVP state. Every step that you are asked to consider or build a new functionality, look in docs/prd for the documentation files and use the content from there. If we make decisions that were not documented in any of the PRD files make sure to update the most suitable file with the decision and if such file doesn't exist consult the user to create one
